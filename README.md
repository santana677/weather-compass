# WeatherComfort

**Интеллектуальный сервис оценки комфортности погоды для планирования пеших прогулок.**

WeatherComfort помогает жителям городов принимать решения о прогулках на основе актуальных погодных данных. Сервис получает прогноз через OpenWeatherMap API, рассчитывает уникальный индекс комфорта (на основе температуры, влажности и ветра) и визуализирует динамику погоды с помощью графиков. Больше никаких неожиданных дождей и промокших ног — только чёткая аналитика!
